- “Fácil de programar”
- Grande Comunidade 
- Gratuito 
- Muitos Recursos
- Frameworks e bibliotecas
- Extremamente Rápido

## Front-End VS Back-end

### Front-End

- Parte da Frente
- Parte Visual 
- O que nosso navegador executa 
- HTML, CSS, JavaScript

### Back-End

- Parte de “Trás”
- Servidor
- PHP, Banco de dados

## Servidor Apache

- Servidor Web
- Protocolo HTTP
- Representa 50% dos servidores WEB
- Windows, Linux, Mac
- Seguro
- Estável

Acessamos um site -> Enviamos para o server 

Navegador <- server
